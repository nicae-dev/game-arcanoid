## Игра арканоид

- Доступна только с компьютера
- Управление ракеткой для ловли шарика через клавиши "влево" и "вправо".
- Можно сбросить рекорд игры и текущий счет кнопкой "Reset score"
- Рекорд игры сохраняется в кеш браузера.
